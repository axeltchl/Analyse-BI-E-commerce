# Analyse-BI-E-commerce

## Description
Projet BI permettant d'analyser les ventes, la compréhension des clients et identification des opportunités d'optimisation


## Fonctionnalités
- Analyse des ventes
- KPIs principaux
- Visualisations interactives
- Analyse selon

## Outils utilisés
- Power BI
- Power Query
- DAX

## Fichiers
- `Dashboard.pbix` : fichier principal Power BI
- `données/` : jeux de données utilisés

## Comment utiliser le projet
1. Télécharger le repository
2. Ouvrir le fichier `.pbix`
3. Reconnecter les sources si nécessaire
