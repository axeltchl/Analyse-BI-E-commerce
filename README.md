# Analyse-BI-E-commerce

## Description
Projet BI permettant d'analyser les ventes, la compréhension des clients et identification des opportunités d'optimisation


## Fonctionnalités
- Analyse des ventes
- KPIs principaux (CA, panier moyen, Taux d'abandaon...)
- Analyse selon différent axes
- Visualisations interactives


## Outils utilisés
- Power BI
- Power Query
- DAX
